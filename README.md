This project involves the development of a complete interactive dashboard using Power BI, covering all stages from data preparation to final visualization. The work is divided into four phases, each contributing to the final analytical solution.


Phase 1 : Basic Data Loading & Cleaning

       Q1. Data Import & Types
Load the dataset into Power BI.
Correct all data types for each column.
Phase 2 : Basic DAX Measures
   
       Q2. Basic Measures
Create measures for:
Total Sales Amount
Total Net Sales (Sales Discount)
Total Order Quantity
Total Shipping Amount
Total Discount Amount
Total Distinct Orders
Total Customers (DISTINCTCOUNT of Customer ID)
Phase 3 : Dashboard Insights

       Q3. Dashboard Visual Questions
       Build visuals to answer:
Which SalesPerson generated the highest Net Sales?
Which Order Priority produces the most revenue?
Which Ship Mode has the highest Shipping Amount?
Which Product Container is used the most?
What is the monthly Sales trend?
Phase 4 : Intermediate DAX

       Q4. Time Intelligence Measures
       Create:
Month-over-Month Growth %
Average Shipping Delay
Weekday wise Sales Pattern
Weeknumber wise Sales Pattern
Salesperson wise Sales contribution on Jumbo Drum
       Q5. Classification Column

       Create a column: Order Value Category
High, Medium, Low (based on Net Sales).
